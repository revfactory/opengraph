# opengraph

## 하네스: OG 추출 기능 개발

**목표:** URL 입력 → Open Graph 추출 기능의 설계·참조구현·운영 체계를 에이전트 팀으로 산출한다.

**트리거:** OG/오픈그래프/링크 미리보기/unfurl/메타데이터 추출 기능 개발, 또는 그 하위 주제(런타임 선택, 동적/리다이렉트/단축링크 크롤링, 실패 추적·실패율 감소, 캐싱 key·DB 선택·응답 속도 아키텍처) 관련 작업 요청 시 `og-extraction-orchestrator` 스킬을 사용하라. 단순 개념 질문은 직접 응답 가능.
